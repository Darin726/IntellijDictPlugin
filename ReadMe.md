###How to use
1. Download the [jar](https://github.com/Darin726/IntellijDictPlugin/blob/master/IntellijDictPlugin.jar)
2. Install the plugin and restart Idea  
![install](http://darin726.qiniudn.com/Snip20170215_4.png)
